## Requirements
- Python 3.6.8
- Flask 1.1.2
- Flask-FlatPages 0.7.2

```pip install flask```    
```pip install Flask-FlatPages```

## Project Structure

```
.
├── main.py
├── pages
│   └── example-text.md
└── templates
    ├── base.html
    └── page.html

```

## Usage

Running ```python main.py```, navigate to the below address in browser:
```
http://localhost:5000
http://localhost:5000/example-text/
```
